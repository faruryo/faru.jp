# faru.jp

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

標準出力に表示される URL でブラウザ確認できる。

だいたい<http://localhost:8080>になっている

### Run your tests

```bash
npm run test
```

### Lints and fixes files

```bash
npm run lint
```

## Project integration

### Docker image build

```bash
docker-compose build
```

### Docker image をローカルで動作確認する

```bash
docker-compose up -d
```

<http://localhost:9000>にアクセスすると動作する。
